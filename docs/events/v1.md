## Core Publisher Foundation (Phase 2)

Core Worker currently publishes:

- `payment_intent.created`
- `quote.created`
- `checkout_session.created`

Event publishing semantics:

- source worker: Core Worker
- envelope format: this document's canonical envelope
- internal transport target: `q-domain-events-{env}` (dedicated domain event stream)
- domain-event publishing is decoupled from delivery command queues
- notification and webhook outbox queues are reserved for delivery-specific commands
- timeline/projection builders can subscribe via dedicated domain-event plumbing in a later phase

## Visibility Rules

Superadmin:
- Full technical detail

Business:
- Financial breakdown
- No internal routing secrets

Consumer:
- Polished status
- Actionable messages
