# Pagooz Event Contracts v1

## Envelope Structure

{
  "event_id": "evt_...",
  "event_type": "payment.succeeded",
  "occurred_at": "ISO_TIMESTAMP",
  "tenant_id": "tnt_...",
  "mode": "sandbox | live",
  "resources": {},
  "data": {},
  "visibility": {
    "superadmin": true,
    "business": true,
    "consumer": false
  }
}

## Visibility Rules

Superadmin:
- Full technical detail

Business:
- Financial breakdown
- No internal routing secrets

Consumer:
- Polished status
- Actionable messages
